Ochrana osobních údajů

Platnost od: 1. dubna 2026  ·  Správce: Adam Štencl

Tato stránka vysvětluje, jaké osobní údaje LifePocket zpracovává, proč a jak dlouho. Aplikace je zdarma, bez reklam a tvá data nejsou nikdy prodávána třetím stranám.

1. Správce údajů

Adam Štencl
E-mail: stencl.adam@gmail.com
Web: lifepocket.app

2. Jaké údaje sbíráme

Při registraci a přihlášení:

  • E-mailová adresa
  • Jméno a profilová fotka (při přihlášení přes Google)

Data která sám zadáváš:

  • Jméno / přezdívka, pohlaví, vybraný avatar
  • Návyky a záznamy o jejich plnění
  • Zápisky a deníkové záznamy (včetně fotografií)
  • Nákupní seznam (osobní i sdílený)
  • Recepty, jídelníček, zásoby
  • Checklist položky
  • Kalendářní události a narozeniny
  • Záznamy o jídle a zdraví (kalorický příjem, voda, nálada)
  • Cíle a vize
  • Konverzace s AI asistentem

Technické údaje:

  • FCM token pro doručení push notifikací
  • Nastavení notifikací
  • Počet AI dotazů (pro rate limiting — ochrana před zneužitím)

3. Proč data zpracováváme

  • Poskytnutí služby — zobrazení tvých dat v aplikaci, synchronizace mezi zařízeními
  • Push notifikace — ranní/večerní připomínky, upozornění na narozeniny
  • AI asistent — kontext konverzace se odesílá ke zpracování Anthropic API (viz níže)
  • Sdílení v rodině — data sdílená se skupinou jsou uložena ve společném Firestore dokumentu přístupném členům skupiny

4. Kde jsou data uložena

Data jsou uložena ve službě Firebase Firestore (Google LLC, servery v EU — region europe-west). Přihlášení zajišťuje Firebase Authentication.

Konverzace s AI asistentem jsou zpracovávány přes Anthropic API (Anthropic PBC, USA). Data se odesílají pouze pro zpracování odpovědi a Anthropic je neukládá déle než 30 dní dle jejich zásad.

5. Jak dlouho data uchováváme

  • Data jsou uložena po dobu trvání tvého účtu
  • Po smazání účtu jsou všechna data trvale odstraněna do 30 dní
  • Záznamy o rate limiting se vymažou automaticky po uplynutí dne

6. Tvá práva (GDPR)

  • Přístup — kdykoli si můžeš stáhnout svá data (Firebase Console nebo na vyžádání)
  • Oprava — údaje lze upravit přímo v nastavení aplikace
  • Výmaz — smazání účtu v nastavení → všechna data budou odstraněna
  • Přenositelnost — na vyžádání zašleme export dat ve strojově čitelném formátu
  • Námitka — máš právo vznést námitku proti zpracování — kontaktuj nás e-mailem

Pro uplatnění práv nebo dotazy piš na stencl.adam@gmail.com. Odpovíme do 30 dní.

7. Soubory cookie

LifePocket nepoužívá sledovací cookies. Využívá výhradně localStorage v prohlížeči pro uložení lokálních nastavení (téma, nastavení notifikací) a Firebase Auth session cookie nezbytnou pro přihlášení.

8. Bezpečnost

Přístup k datům je chráněn Firebase Authentication. Každý uživatel vidí pouze svá vlastní data — sdílená data jsou přístupná jen členům dané skupiny. API klíče jsou uloženy na serveru a nikdy nejsou odesílány do prohlížeče.

9. Změny těchto zásad

O podstatných změnách budeme informovat prostřednictvím aplikace (sekce Co je nového). Datum poslední aktualizace je uvedeno v záhlaví tohoto dokumentu.
